# Heart Disease Risk Analysis Agent

## Overview
This project is an AI-powered Heart Disease Risk Analysis Agent built using n8n, Google Gemini AI, and Google Sheets.

The workflow automatically reads patient health records, identifies risk factors, classifies risk levels, and generates recommendations.

## Features
- Patient Health Record Analysis
- Heart Disease Risk Detection
- Risk Level Classification (Low, Medium, High)
- AI-Generated Recommendations
- Automated Workflow using n8n

## Tech Stack
- n8n
- Google Gemini AI
- Google Sheets
- Python

## Workflow Architecture

Google Sheets
        ↓
    AI Agent
        ↓
 Google Gemini AI
        ↓
 Risk Assessment Report
